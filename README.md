# ufsc-licence-competition
Plugin WordPress add-on pour l’UFSC permettant la gestion des licences officielles, la synchronisation ASPTT, les licences PDF nominatives, l’accès clubs et l’évolution vers la gestion des compétitions (championnats, open, galas).

## Shortcodes

Shortcodes confirmés à documenter :

- `ufsc_lc_club_licences_asptt` — shortcode principal pour l’accès aux licences club ASPTT.
- `ufsc_lc_licences` — alias safe préfixé pour la même vue.
- `ufsc_club_licences_asptt` — alias legacy (uniquement si le mode legacy est activé).
- `ufsc_licences` — alias legacy générique (uniquement si le mode legacy est activé).
